# Contact Page
A simple and responsive Contact Page built using modern front-end technologies. 
This project demonstrates I created a simple, yet clean interface for collecting contact information.

## Features
- Responsive design
- Input fields for user details (e.g., name, email, message)
- Easy to customize and integrate into any website

## Technologies Used
- HTML 
- CSS
- PHP

## How It Works
1. Users fill out the contact form with their details.
2. The form validates the input fields.
3. Upon submission, the data can be:
   - Sent to a backend (if integrated)
   - Logged or handled via JavaScript
